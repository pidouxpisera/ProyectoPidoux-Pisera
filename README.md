# 🧠 Analizador de Perfil de Estudio Híbrido

[![Deploy to GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Deployed-brightgreen)](https://pidouxpisera.github.io/)

Test psicopedagógico interactivo que descubre tu combinación exacta de canales de aprendizaje, administración del tiempo y perfil cognitivo social.

## 🚀 Acceso

**[Abre el Analizador en vivo](https://pidouxpisera.github.io/)**

## ✨ Características

### 🧠 Análisis Multidimensional
- **Canales VARK**: Visual, Auditivo, Kinestésico, Lectoescritura
- **Perfil Psicológico**: Introversión vs. Extroversión
- **Gestión del Tiempo**: Organización, Procrastinación, Enfoque

### 🎨 Algoritmo Inteligente
- Cálculo porcentual automático de canales de aprendizaje
- Motor de lógica condicional que genera sesiones personalizadas
- Recomendaciones contextuales basadas en tu perfil

### ⚡ Tecnología
- **Código nativo**: HTML5, CSS3, JavaScript vanilla
- **Sin dependencias**: Sin librerías externas lentas
- **Optimizado para SEO**: Metaetiquetas OG, descripciones y viewport
- **Responsive**: Funciona perfectamente en desktop, tablet y móvil

## 📋 Estructura del Test

### Bloque 1: Canales de Procesamiento (4 preguntas)
1. Preferencia visual (diagramas, gráficos)
2. Preferencia auditiva (explicaciones, podcasts)
3. Preferencia kinestésica (aprendizaje práctico)
4. Preferencia lectoescrita (notas, transcripción)

### Bloque 2: Perfil Psicológico y Social (2 preguntas)
5. Extroversión en estudio grupal
6. Introversión en análisis solitario

### Bloque 3: Gestión del Tiempo y Foco (3 preguntas)
7. Planificación y estructura
8. Procrastinación y presión
9. Capacidad de atención sostenida

## 📊 Resultados Personalizados

En función de tus respuestas, recibirás:

✅ **Distribución de Canales**: Porcentaje exacto de cada estilo de aprendizaje

✅ **Sesión de Estudio Ideal** con 4 componentes:
1. **El Escenario Ideal**: Ambiente recomendado (aislado vs. dinámico)
2. **El Arranque**: Preparación y establecimiento de objetivos
3. **El Núcleo**: Técnica específica según tu canal dominante + ritmo de descansos
4. **Consolidación**: Cierre y retención de información

## 🏆 Técnicas Incluidas

- **Mapas Conceptuales**: Para aprendices visuales
- **Evocación Verbal**: Para aprendices auditivos
- **Kinestesia Activa**: Para aprendices kinestésicos
- **Método Cornell**: Para aprendices de lectoescritura
- **Pomodoro Adaptado**: Para baja tolerancia al enfoque
- **Deep Work**: Para alta capacidad de concentración sostenida
